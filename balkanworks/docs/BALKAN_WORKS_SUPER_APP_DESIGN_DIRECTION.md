# Balkan.works Super App Design Direction

The web implementation follows an AI-first, mobile-first premium-minimal direction. Home begins with intent-based AI search, discovery uses visual category cards and a personalized local feed, and each business profile presents trust, actions, services, reviews and an AI summary as one cohesive experience.

## Interaction priorities

1. A user can express a need before knowing a category or business name.
2. The central AI navigation action stays distinct from standard navigation.
3. Trust is visible through verification, review count, quality score and practical action buttons.
4. Dashboard and administration remain calm, information-led operational surfaces.

Demo visuals are deliberately abstract until businesses supply licensed profile and cover imagery. API-backed image fields replace those placeholders without changing the component contracts.
