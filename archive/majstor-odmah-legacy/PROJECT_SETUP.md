# Balkan.works Project Setup

The target project setup is implemented in `balkanworks/`: Turborepo, TypeScript, Next.js web/admin, React Native mobile, NestJS API, Prisma/PostgreSQL, Redis/S3-ready environment and shared packages. Environment values are only represented in `.env.example`; no secret has been copied.

The initial milestone is Balkan ID, profile, business creation, business locations and roles. The production Python Core remains separate until a reviewed migration; it is not overwritten by this new foundation.
