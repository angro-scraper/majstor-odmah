# Arhiva prethodnog projekta

`majstor-odmah-legacy/` sadrži prethodnu Majstor Odmah aplikaciju i prateće prototipe, sačuvane prilikom prelaska na Balkan.works 18. jula 2026.

Ovaj sadržaj nije deo aktivnog Balkan.works builda ni novog Render deploymenta. Čuva se isključivo radi istorije, referenci i eventualne migracije podataka.
